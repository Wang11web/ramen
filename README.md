# Vue.js 電商開發 - ramen

## 說明

此作品用 Vue 開發，重新設計製作知名烏龍麵品牌網頁，使用者可以選擇欲購買產品，並於購物畫面進行結帳，另有後臺介面，管理者可做商品品項的增減以及使用者的結帳狀態等。

### 作品網址

[網址](https://wang11web.github.io/ramen/)


### 功能說明
#### 前台
* 使用 Bootstrap5 + Sass 切版
* 套用 AOS 動畫
* 串接遠端 API 
* 購物車系統
* 使用 vee-validation 做表單驗證
